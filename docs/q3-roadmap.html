Monetization

Plugin + review stay free. The GitHub App carries the org paywall (private repos, reviewers, audit). The 3-diagram limit is the in-editor upgrade moment. Never gate the magic moment — gate the systems that scale it.

Dependencies & risks

  • Measurement (PLUG-98) gates the whole bet — ship it with v2, not after.
  • ~1 eng (Prashant), no Plugins PM — themes run in sequence, not parallel.
  • OSS sensitivity — keep the public-repo path generous; gate on private / org.
